# Dental AI System

A full-stack dental appointment management platform designed to streamline appointment scheduling and clinic operations.

## Features

- Patient, Doctor, and Nurse roles
- JWT-based Authentication and Authorization
- Role-Based Access Control (RBAC)
- Appointment Booking
- Appointment Cancellation and Rescheduling
- AI Chatbot for Appointment Scheduling
- Secure Password Hashing with bcrypt
- REST APIs built with FastAPI
- PostgreSQL Database Integration
- Audit Logging and Activity Tracking

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- JWT Authentication
- Passlib (bcrypt)

### Frontend
- HTML
- CSS
- JavaScript

### DevOps
- Git
- GitHub
- Environment Variables (.env)

## Project Goal

This project demonstrates practical full-stack software engineering skills, including API development, authentication, database design, role-based security, chatbot integration, and modern deployment practices.
